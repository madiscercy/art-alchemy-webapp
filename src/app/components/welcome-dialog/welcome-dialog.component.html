Welcome to Art Alchemy!

Art Alchemy is my portfolio project that is currently under active development. I'm working diligently to complete it as soon as possible.

The project is built with Angular and hosted on an AWS S3 bucket. It makes calls to an AWS Python Lambda function which integrates with the DeviantArt API.

I appreciate your understanding and invite you to revisit the site to witness its evolution. In the meantime, I encourage you to explore what's already here and check back frequently for new updates.
